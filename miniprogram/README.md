# 拓界足迹微信小程序

这是现有网页旅行地图的原生微信小程序版本。当前已实现功能全部直接开放使用。

## 当前功能

- 原生微信地图
- 全球国家边界显示与点亮
- 中国 34 个省级行政区边界显示与点亮
- 按省显示中国市级边界并点亮
- 国家、省级、市级地图层级切换
- 地图打卡标记
- 旅行总览和五类核心成就等级
- 不限数量的旅行打卡
- 359 个中国 5A 景区清单
- 1248 个世界遗产清单
- 清单搜索、分页和进度统计
- 本地数据持久化
- 剪贴板备份导入与导出
- 相册照片选择入口

## 运行

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择本目录 `miniprogram`。
4. 暂时使用测试号，或把 `project.config.json` 中的 `appid` 替换为正式小程序 AppID。

## 数据生成

地图边界和清单来自仓库现有数据。源数据更新后，重新运行：

```powershell
node tools/build-miniprogram-boundaries.js
node tools/build-miniprogram-checklists.js
```

生成的小程序数据文件位于 `miniprogram/data`。
